## Git
概念：是一个分布式的代码管理工具

优势

  ```
  1.没有单点故障
  2.支持分布式部署，本地其实就是一个本地仓库，支持离线提交，相对独立，不影响开发
  3.更少的仓库污染
  4.各大管理网站都用git
  ```
